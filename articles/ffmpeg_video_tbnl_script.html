Windows / Video thumbnails script with FFmpeg

 Simple script that uses FFmpeg and extracts pointed to frames to jpg files. In my case, the names and dates of the video files were lost, but every clip had visible film slate during the first seconds so it was possible to dump timeframe 00:00:00 and comfortably restore missing names and include dates. To read more about cmd details please reffer to this article: 7zip batch extract to subfolders / DE Version.
SET source=C:\Users\Bill\Videos\ SET ffmpeg=C:\Users\Bill\Downloads\ffmpeg\ffmpeg-6.0-full_build\bin\ffmpeg.exe SET destination=C:\Users\Bill\Videos\tbnls\ FOR /F "tokens=*" %F IN ('DIR /S /B "%source%\*.MKV"') DO %ffmpeg% -i "%F" -ss 00:00:00 -vframes 1 -s 1920x1080 "%destination%%~nxF.jpg"
